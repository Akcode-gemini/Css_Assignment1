Positioning Property

1.static

Static property goes with the normal flow of the page.Properties like left,right,top,bottom and z-index do not affect an element with position:static

2.relative

Relative property also goes with the normal flow of the page.Properties like left,right,top,bottom and z-index affect an element with position:relative as shown in the output.

3.absolute

Absolute property does not go with the normal flow of the page.Properties like left,right,top,bottom and z-index determine the final position of an element with position:absolute. With this property element is positioned relative to its closest positioned ancestor other that element having position:static

4.Fixed

Fixed property also does not go with the normal flow of the page.Properties like left,right,top,bottom and z-index determine the final position of an element with position:fixed. With this property element is positioned relative to its html element.They are not affected by scrolling.
